Ordered list of common HTML tags along with some information

  1. html: The root element that wraps all the content on an HTML page.
  2. head: Contains meta-information about the document, such as the title and links to external resources.
  3. title: Sets the title of the HTML document, which appears in the browser's title bar or tab.
  4. meta: Provides metadata about the HTML document, such as character encoding or authorship information.
  5. link: Used to link external resources like stylesheets to the HTML document.
  6. style: Defines CSS styles directly within the HTML document.
  7. script: Embeds or references JavaScript code within the HTML document.
  8. body: Contains the visible content of the HTML page, including text, images, and other elements.
  9. header: Represents introductory content typically found at the top of a webpage, such as logos and navigation menus.
  10. nav: Defines a section of navigation links.
  11. main: Represents the main content area of the document.
  12. article: Represents a self-contained composition within a document, such as a blog post or news article.
  13. section: Defines a thematic grouping of content within an HTML document.
  14. aside: Contains content that is tangentially related to the surrounding content.
  15. footer: Represents the footer of a section or the entire document, often containing copyright and contact information.
  16. ( h1, h2, h3, h4, h5, h6): Heading elements, used to define hierarchical headings from most important (h1) to least important (h6).
  17. p: Represents a paragraph of text.
  18. a: Creates hyperlinks to other web pages or resources.
  19. img: Embeds images within the document.
  20. ul: Defines an unordered (bulleted) list.
  21. ol: Defines an ordered (numbered) list.
  22. li: Represents a list item within ul or ol.
  23. div: A generic container element often used for layout and styling purposes.
  24. span: A generic inline container often used for styling or scripting individual pieces of text.
